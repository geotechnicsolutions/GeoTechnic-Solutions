# Geotechnic Solutions

This repository contains the static website for Geotechnic Solutions, a geoscience consultancy for the green transition. Geotechnic Solutions offers early-stage geothermal screening, critical minerals assessments, and AI/GIS-supported feasibility studies.

The website is a single-page static site built using HTML and CSS (no framework). To make local changes, edit the `index.html` file. To deploy, host the contents of this repository on a static hosting platform (e.g. Vercel, Netlify, GitHub Pages).

For more information, visit [geotechnic.solutions](https://www.geotechnic.solutions).
